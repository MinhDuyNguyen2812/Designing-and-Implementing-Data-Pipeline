Overview
This repository contains a structured progression of Python development, ranging from foundational Object-Oriented Programming (OOP) to functional desktop applications and data pipeline implementations.

A desktop-based task management system designed for corporate environments.

Technology Stack: Python, Tkinter, SQLite.

Core Functionality: User authentication, task CRUD (Create, Read, Update, Delete) operations, and persistent data storage.

Location: projects/company_task_app/
